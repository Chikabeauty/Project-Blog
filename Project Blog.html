My First Steps in Coding

Date: September 28, 2024

Today marks the beginning of my journey into programming. I started learning the basics of HTML and CSS, and it has been both challenging and exciting. I am amazed at how much you can create with just text and simple commands.

Learning HTML felt like a natural introduction to the world of programming because it's all about structuring content. CSS is equally fascinating, as it allows you to give style to your HTML content and make it visually appealing.

I can't wait to dive deeper into these technologies and explore more complex projects!

Understanding HTML and CSS

Date: November 7, 2024

This week, I really focused on understanding HTML and CSS in more detail. I learned how to use different tags to structure a webpage, and how CSS allows you to manipulate layout, colors, fonts, and more.

One of the challenges I faced was learning about CSS, but I feel much more confident now that I understand how margin, padding, and borders work.

As I build more projects, I want to experiment with responsive design, so that my webpages can look good on both desktops and mobile devices.
